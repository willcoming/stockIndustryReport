1. 一頁式懶人包

  • 公司一句話：Sunrun 是美國住宅太陽能、家用電池及家戶至電網（VPP）服務商，以訂閱/租賃和系統銷售建立長期合約現金流。
  • 所屬產業：分散式能源、住宅太陽能、儲能與虛擬電廠；位於設備供應商與家庭/電網之間的整合服務層。
  • 核心邏輯：Q1 收入與 GAAP 獲利改善、儲能附掛率升至 73%，但新增用戶及安裝容量下滑、專案融資與政策仍是高敏感變數。
  • 評級：觀察。來源列「現在不能買但值得等買點」，LZEMA20 技術轉折不足以抵銷融資、法規和執行風險。
  • 追蹤：新增用戶、儲能附掛率、Contracted Net Value Creation、現金生成、母公司追索債、ITC/淨計量政策。

2. 公司如何賺錢

  • 向住戶提供太陽能+儲能系統，以訂閱、租賃、電力服務協議及產品/系統銷售取得收入；另透過聚合電池容量參與電網服務。
  • Q1 客戶協議及 incentives 收入 4.678 億、系統/產品銷售 2.544 億。後者受 2025Q3 的資產出售結構影響，不能把單季營收增幅直接外推。
  • 客戶為美國住宅用戶、融資夥伴及電網/能源市場；資本密集、長約與稅賦抵減使現金流時點比 GAAP 利潤更重要。

3. 從公司反推產業

  • 分散式能源解決家庭備援、電費管理與電網尖峰容量需求；電池讓住宅太陽能由單純發電轉為可調度資產。
  • 產業仍在成長但政策驅動與利率敏感，導入速度受電網併網、地方許可、融資成本和消費者經濟性制約。
  • 長期驅動：電網韌性、儲能成本下降、VPP；短期變數：聯邦 ITC、州淨計量、利率、模組/電池供應和融資市場。

5. 商業模式與護城河

  • 規模：2026Q1 末訂戶 1,014,945、年增 11%；已安裝逾 251,000 個儲能/太陽能系統、約 4.3GWh 網路化儲能。
  • 產品：Sunrun Flex 以可擴張用電需求的太陽能+儲能訂閱拉高用戶價值；Q1 Flex mix 年比逾倍。
  • 護城河是通路、用戶資料、施工/服務能力、專案融資與 VPP 聚合；但無法消除政策與資本成本波動。

6. 重要財務與營運指標

  • 營收：2026Q1 7.222 億美元、年增 43%；應分拆客戶協議/產品交易結構解讀。
  • 交付：新增訂戶 17,665、年減 25%；太陽能裝置 154MW、年減 19%，儲能 282MWh、年減 15%。
  • 單位經濟：儲能附掛率 73%；Net Subscriber Value 11,892 美元、年增 14%；Contracted Net Value Creation 1.08 億、年減 34%。
  • 現金與槓桿：Q1 Cash Generation -5,900 萬（排除 safe-harbor 投資後 -3,100 萬）；公司維持全年 2.5–4.5 億指引，並稱已償還 9,200 萬追索債、2028/3 前無追索債到期。

11. 投資標的分層

  • 核心穩健型：尚非，資本與政策敏感度高。

  • 景氣循環型：利率和住宅需求主導。
  • 高風險高報酬型：是，專案融資可放大上/下行。
  • 只適合觀察：現階段適用。

12. 產業風險與反方觀點

  • 政策：ITC、關稅、淨計量與州規則改變可壓縮需求/回報。
  • 融資：資金成本、稅權益投資者與資產證券化市場若收縮，會限制成長。
  • 營運：獲客成本、施工、電網併網、設備保固及極端天氣。
  • 警訊：新增訂戶/容量續減、Contracted Net Value 下降、Cash Generation 未達指引、追索債或稀釋增加。

14. 結論

  • Q1 獲利與儲能指標具正面訊號，但現金生成、容量新增與政策敏感度構成反方論點。
