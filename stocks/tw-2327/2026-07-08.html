1. 一頁式懶人包

  • 公司一句話介紹：國巨是全球被動元件龍頭之一，產品涵蓋 MLCC、晶片電阻、電感、鉭質電容、磁性與感測元件。
  • 所屬主要產業：被動元件。
  • 產業鏈位置：中游被動元件與高可靠度電子零組件供應商，服務車用、工業、通訊、AI server、消費電子與通路客戶。
  • 核心投資邏輯：公司具全球規模、產品組合與車工規客戶黏著度，2026 年上半年營收年增 29.34%；但 P/E 71.26、P/B 11.15 顯示市場已大幅預支 AI/車用與高階被動元件成長。
  • 目前投資評級：觀察。
  • 主要理由：基本面品質佳，但 workorder 來源報告將其列入「明確不可以買 / 暫時排除名單」，短線跌破月線且估值偏高，新資金缺乏安全邊際。
  • 主要風險：被動元件報價循環、AI server 題材過度反映、庫存反轉、併購整合與匯率、標準品競爭。
  • 最重要的 3-5 個追蹤指標：月營收、毛利率、營益率、EPS、庫存週轉、車用/AI server 高階產品占比。
  • 代表性美股與台股同業：Murata、TDK、Vishay、Kyocera；台股華新科、立隆電、大毅、信昌電、光頡、華容。

2. 公司如何賺錢

  • 主要產品或服務：MLCC、晶片電阻、電感、鉭質電容、保護元件、磁性元件、感測器與特殊電子零組件。
  • 營收來源：向 EMS、車用、工業、網通、AI server、消費電子與通路客戶銷售被動元件與高可靠度零組件。
  • 主要客戶：官方未在免費端點完整揭露客戶名單；應用端涵蓋汽車、工業、通訊、消費電子與伺服器供應鏈，客戶集中度需查證。
  • 毛利率與營益率特性：2026Q1 毛利率約 38.1%、營益率約 25.2%。高階車用/工業/特殊規格毛利較佳，標準品受供需與中國廠競爭影響較大。
  • 成長主要來自價格、出貨量、市占率、新產品，還是產業總需求擴張：2026 年 6 月營收 153.59 億元、年增 38.91%；1-6 月累計營收 826.22 億元、年增 29.34%。成長來自高階產品 mix、AI/車用需求、併購產品線與報價循環共同推動。
  • 是否有景氣循環性：有。被動元件是成熟週期產業，庫存與報價會放大 EPS 波動。

3. 從公司反推產業

  • 公司所屬產業：被動元件與高可靠度電子零組件。
  • 產業解決什麼問題：被動元件提供儲能、濾波、穩壓、抗突波、訊號耦合、電磁干擾抑制，是所有電子系統不可或缺的基礎零件。
  • 需求來自哪裡：AI server 電源密度提升、車用電子含量增加、工業自動化、通訊設備、PC/手機與消費電子補庫存。
  • 產業目前處於導入期、成長期、成熟期，還是景氣循環期：成熟且週期性高；AI server、車用與工業高可靠度產品是成長支線。
  • 長期成長驅動力：電動車與 ADAS、AI server、工業自動化、高壓高容 MLCC、能源與電源管理需求。
  • 短期波動因素：通路庫存、報價、終端拉貨、匯率、同業擴產、題材資金與技術面。

5. 商業模式與護城河

  • 技術壁壘：高階 MLCC、車用與工業規格、可靠度認證、材料與製程管理形成壁壘。
  • 規模經濟：全球產能、採購、通路與併購整合使國巨具成本與產品線規模。
  • 客戶黏著度：車用、工業與高可靠度客戶驗證期長，導入後切換成本較高。
  • 轉換成本：高階產品轉換成本高；一般標準品轉換成本低。
  • 品牌或通路優勢：國巨、KEMET、Pulse 等產品組合提升全球客戶覆蓋。
  • 成本優勢：來自規模採購、產能配置與產品 mix，但毛利率仍需連續追蹤。
  • 專利、法規或資本門檻：車規、工規、品質系統、資本支出與全球認證形成門檻。
  • 是否容易被替代：標準品容易被低成本廠替代；車用/工業高階規格替代性較低。

  • 高風險高報酬型：目前符合。高估值與短線弱勢會放大波動。

12. 產業風險與反方觀點

  • 估值過高風險：P/E 71.26、P/B 11.15 對成熟週期產業容錯率低。
  • 景氣反轉風險：被動元件缺貨漲價後常出現庫存反轉。
  • 技術替代風險：系統設計、模組化與低成本替代供應可能壓縮標準品利潤。
  • 競爭加劇風險：中國與亞洲同業在標準品擴產，可能降低報價。
  • 政策與地緣政治風險：車用、AI server 與跨國併購產品受關稅、出口限制與匯率影響。
  • 客戶集中風險：若 AI/車用大客戶拉貨放緩，成長預期會快速修正。
  • 財報可能出現的警訊：營收成長但毛利率未升、庫存增加、應收帳款拉長、自由現金流弱於淨利。

  • 該產業是否值得長期追蹤：值得。AI、車用與工業電子化會增加高階被動元件需求。
  • 目前最重要的投資邏輯：高階產品 mix 與全球規模能否讓 EPS 追上高估值。
  • 最大風險：估值已反映太多樂觀預期，若報價或毛利率不如預期，股價容易修正。
  • 估值是否已經反映成長預期：大多已反映。P/E 71.26 需要未來 EPS 大幅上修才合理。
