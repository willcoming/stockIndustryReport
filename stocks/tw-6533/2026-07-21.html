1. 一頁式懶人包

  • 公司一句話：晶心科是專注處理器核心與軟體工具鏈的矽智財公司，主力由自研 AndesCore／AndeStar 延伸至 RISC-V CPU IP。
  • 所屬主要產業：半導體矽智財（處理器 IP）；產業鏈位置在晶片架構、CPU 核心授權與軟體生態，位於 IC 設計之前端關鍵層。
  • 核心投資邏輯：RISC-V、AI 加速器、車用與資料中心帶動授權案；一旦客戶量產，權利金可形成長尾收入，但研發前置支出高、案件與認列時點不均。
  • 目前投資評級：觀察。
  • 主要理由：2026 上半年營收明顯反彈，卻仍需等待 Q2 驗證損益轉折；2026 Q1 仍虧損，2025 全年亦淨損 4.153 億元。7/20 股價單日漲停至 231 元且估值應以 P/S、P/B 與現金消耗檢視，安全邊際尚不充分。
  • 主要風險：高研發費用、客戶 tape-out／量產延期、Arm 與其他 RISC-V IP 競爭、出口管制、人才成本、匯率及收入高度季節性。
  • 最重要追蹤指標：IP 授權收入、權利金收入、AI／車用占比、量產 SoC 累計出貨、營業費用／營收、營業現金流與 Q2 是否縮小虧損。
  • 代表同業：美股 ARM、Synopsys、Cadence；台股 智原、創意、力旺、世芯-KY、聯發科。

2. 公司如何賺錢

  • 矽智財授權：客戶在晶片設計階段取得 CPU 核心、延伸指令與開發工具授權，收入多與合約里程碑相關。2025 年報揭露授權收入 9.654 億元、占 65%。
  • 權利金：客戶晶片量產後依出貨或約定支付。2025 年權利金 2.823 億元、占 19%，是商業模式能否由專案轉成複利收入的關鍵。
  • 維護與服務：2025 年維護服務收入 2.270 億元、占 16% 左右，支撐客戶整合、軟體與後續版本。
  • 公司毛利率接近 100% 不代表高淨利率；主要成本在研發工程師、EDA／開發環境、海外據點與銷售。2026 Q1 毛利率約 99.9%，但營業費用 5.289 億元高於營收，導致營業虧損。
  • 成長來源是授權案數與單價、客戶量產率、每顆權利金及 AI／車用高階核心滲透；公司不製造晶片，沒有傳統製造產能利用率，但有研發人力利用率與專案交付風險。

3. 從公司反推產業

  • 處理器 IP 解決「客戶不必從零設計 CPU」的問題，縮短上市時間並降低驗證風險；RISC-V 的開放 ISA 讓客戶可客製化，但商用核心、工具鏈與支援仍可收費。
  • 需求來自 AI 加速器控制核、車用 MCU／ADAS、資料中心、網通、儲存、IoT 與消費電子 SoC。
  • 產業處於成長期，但商業化仍有長週期：授權到 tape-out、再到量產與權利金，可能跨越數季至數年。
  • 長期驅動力：AI 專用晶片增加、晶片自主化、RISC-V 生態成熟、客製運算與車用功能安全。
  • 短期波動：大型授權案驗收、客戶融資與量產時程、出口限制、研發費用及匯率。

5. 商業模式與護城河

  • 技術壁壘：處理器微架構、向量運算、功能安全、低功耗、驗證與軟體工具鏈需要多年累積。
  • 轉換成本：CPU 核心一旦整合進 SoC，軟體、驗證與認證形成黏著；但新專案仍可能重新評估 Arm 或其他 RISC-V 供應商。
  • 生態系：編譯器、除錯器、開發板、作業系統與第三方 IP 的成熟度，會直接影響客戶採用。
  • 商業優勢：純 IP、不與客戶推出自有晶片競爭；2025 年報稱累計客戶 SoC 出貨超過 190 億顆，顯示實際落地基礎。
  • 弱點：規模、品牌與開發者生態仍小於 Arm；高階核心須持續投入，短期費用可能快於權利金成長。

6. 重要財務與營運指標

  • 營收成長：2025 年營收 14.782 億元、年增約 7%；2026 H1 8.926 億元、年增 62.07%。需拆解授權、權利金與服務，避免被單月里程碑誤導。
  • 毛利率／營益率：Q1 毛利率約 99.9%，營益率約 -62.2%；改善要靠收入規模跨過固定研發費用門檻。
  • EPS：2025 EPS -8.20 元；2026 Q1 EPS -3.28 元。轉盈前 P/E 不適用。
  • 自由現金流：2025 年營業現金流入約 3.099 億元，但投資現金流出約 24.456 億元、籌資流入約 14.740 億元；大型投資內容與可持續現金消耗需查證。
  • 資本支出／無形資產：IP 公司傳統廠房 CapEx 輕，但研發與可能取得無形資產仍需納入經濟成本。
  • 存貨與產能利用率：非核心指標；更應追蹤授權 backlog、tape-out、量產客戶、權利金與研發人效。
  • 客戶集中度／市占率：需由 2025 年報附表再次查證；單一大型客戶驗收可能顯著改變季度收入。

11. 投資標的分層

  • 核心穩健型：目前不屬於；盈餘與現金流尚未具穩定性。
  • 成長型：屬於，成長來源是 RISC-V、AI、車用與客製運算。
  • 景氣循環型：部分屬於，客戶投片與半導體資本循環會影響授權和量產。
  • 高風險高報酬型：屬於；固定研發成本造成營運槓桿，轉盈或失望都可能放大股價反應。
  • 只適合觀察、不適合輕易買進：現階段最合適分類，直到獲利拐點與估值安全邊際同時出現。

12. 產業風險與反方觀點

  • 多頭認為開放 ISA 與 AI 客製晶片會擴張 RISC-V 市場；反方認為開放不等於晶心必然取得商業價值，Arm 生態與其他 RISC-V 廠商仍可競爭。
  • 估值風險：虧損公司在題材熱時容易以遠期營收定價，若量產延後，倍數可快速收縮。
  • 技術替代：客戶自研核心、Arm、SiFive／其他 RISC-V 核心，以及不同加速器架構。
  • 政策與地緣：出口管制、客戶所在地限制與匯率，都可能改變成交與收款。
  • 財報警訊：授權增但權利金不增、營收成長卻應收／合約資產惡化、費用增速持續高於營收、現金消耗擴大。

14. 結論

晶心科是理解 RISC-V 處理器 IP 的好入口，產業長期值得追蹤，核心邏輯是授權案最終轉成可重複權利金。2026 上半年營收反彈是真實正面訊號，但 Q1 仍大幅虧損、7/20 股價急漲，估值已反映部分成長期待。最大風險是高研發支出先發生、客戶量產與權利金延後。最終評級為觀察；下一步優先研究高階 CPU 核心量產、AI／車用客戶、軟體生態與權利金轉換。本報告不構成個人化投資建議或獲利保證。
