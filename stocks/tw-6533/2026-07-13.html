1. 一頁式懶人包

  • 公司一句話介紹： 晶心科為 32/64 位元嵌入式處理器 IP 與 SoC 設計平台公司，主力是 AndesCore RISC-V CPU IP、開發工具及客製化服務。
  • 產業／位置： 半導體 IP 供應鏈中游，將 CPU 核心／工具授權給 IC 設計與系統公司，最終應用在 AI、車用、網通、儲存、IoT 和消費電子。
  • 核心投資邏輯： IP 授權具有高毛利、遞延權利金與生態系效應；但 2026Q1 仍虧損 EPS -3.28 元，6 月營收因 IP 授權年增 247.0%，反映收入認列高度集中，不能只以單月判定獲利反轉。
  • 目前投資評級：觀察。 RISC-V 長期滲透與公司技術地位值得追蹤，但虧損、營收認列波動與估值無 P/E 錨點，須先看到連續的本業改善。
  • 主要風險： 授權／權利金集中與認列時點、RISC-V 生態競爭、研發費用、客戶產品量產延遲，以及可轉換性金融負債／資本結構的理解風險。
  • 追蹤指標： 新授權與量產權利金、季度營收與營益率、研發費用率、現金流、主要客戶／應用分布、AndesCore SoC 累積出貨與生態合作進度。
  • 代表同業： 台股 M31（6643）、力旺（3529）、創意（3443）、世芯-KY（3661）；美股 Synopsys（SNPS）、Cadence（CDNS）、Arm（ARM）、CEVA（CEVA）。

2. 公司如何賺錢

  • 主要產品／服務： CPU IP 軟核／硬核授權、AndeSight 開發工具、平台與技術服務，並在客戶晶片量產後收取權利金；年報對兩類授權機制與維護服務有說明。
  • 客戶： IC 設計、SoC、嵌入式系統與平台公司。客戶名稱、前五大占比和個別合約金額並非本研究可驗證公開資料，均標示 需查證。
  • 毛利／營益特性： 2026Q1 營收 3.264 億元、毛利率約 99.9%，但研發與營業費用 5.289 億元，使營業損失約 2.030 億元；這是典型 IP 公司的高毛利、重研發模型。
  • 成長來源： RISC-V 滲透、AI／向量／車規安全核心、客製擴充、開發工具和客戶量產權利金。
  • 循環性： 授權簽約和客戶量產節點會造成季度跳動，半導體資本支出與設計活動景氣亦會影響新案。

3. 從公司反推產業

  • 產業功能： 將 CPU 核心、軟體工具與技術支援標準化，讓客戶能更快完成 SoC，且可依應用做客製化擴充。
  • 需求來源： AI 邊緣運算、車用電子、工控、儲存、網通、IoT、消費電子與資料中心等 SoC 設計活動。
  • 產業階段： 處理器 IP 是成熟且高門檻產業；RISC-V 開放指令集和 AI／車用需求仍是成長階段，但生態成熟度與商業化速度不一。
  • 長期／短期驅動： 長期取決於 RISC-V 生態、客製指令與軟體工具；短期則受客戶簽約、設計定案、tape-out 與量產權利金認列影響。

5. 商業模式與護城河

  • 技術壁壘： 官方資料顯示其提供可配置 32/64 位元核心，涵蓋 DSP、FPU、向量、超純量與多核心；CPU 驗證、軟體相容和客戶支援需長期投入。
  • 生態與黏著： AndesCore、AndeSight 與合作夥伴形成設計流程黏著；官方於 2023 年底統計 CPU IP 授權逾 300 個、累積 SoC 出貨逾 140 億顆，後續數字與計算口徑需查證。
  • 轉換成本： 客戶完成 CPU 選型、SoC 整合、軟硬體驗證後轉換有成本，但 RISC-V 的開放性也降低指令集本身的鎖定，競爭優勢須來自核心品質、工具和服務。
  • 替代性： ARM、Synopsys、Cadence、CEVA 與其他 RISC-V 供應商皆可競爭；大客戶自行設計核心亦是風險。

11. 投資標的分層

  • 核心穩健型： 具較成熟權利金與獲利紀錄的 IP／EDA 公司可作基準，實際估值仍須查證。
  • 成長型： 晶心科屬 RISC-V、AI 與車用成長型，但回報取決於設計導入能否量產。
  • 景氣循環型： 客戶 SoC 設計活動與半導體 CAPEX 會影響新授權。
  • 高風險高報酬型： 收入認列波動、先行研發投資與轉盈不確定使其符合此類。
  • 只適合觀察： 對虧損公司估值與 IP 商業模式不熟悉者，現階段宜觀察。

12. 產業風險與反方觀點

  • RISC-V 成長不等於所有 IP 供應商都能取得定價權；開放 ISA 也可能增加競爭。
  • 高毛利不能掩蓋營業費用和虧損，轉盈的時間與幅度是估值核心。
  • 單月授權收入可能高度集中，客戶量產延後會造成季度大幅變動。
  • 大型 EDA／IP 業者、Arm 與客戶內製能力可能壓縮差異化；車用／AI 認證期亦較長。

14. 結論

晶心科是理解 RISC-V 商用 CPU IP 與台灣 IP 生態的好入口。官方資料展現其核心、工具與合作生態，且 2026H1 營收明顯成長；但是 Q1 仍為營業虧損，6 月的授權拉升尚須用連續季度和現金流驗證。因此最終評級為 觀察。最重要的研究不是單看高毛利或題材，而是追蹤授權能否轉化為可重複權利金、營業獲利與健康現金流。
