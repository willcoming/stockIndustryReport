1. 一頁式懶人包

  • 公司一句話介紹：力旺是嵌入式非揮發性記憶體矽智財公司，靠授權 IP、技術服務與量產權利金收費。
  • 所屬主要產業：ASIC 客製化晶片與矽智財，細分為嵌入式 NVM IP、資安 IP 與半導體授權服務。
  • 產業鏈位置：半導體設計上游 IP 供應商，位於晶片設計、晶圓代工與 ASIC 設計服務之前段。
  • 核心投資邏輯：商業模式具高毛利、低資本支出與 royalty 放大效果；AI、車用、MCU、先進製程與安全 IP 需求提供長線成長，但 2026-07-07 P/E 100.00、P/B 46.10 已反映高度成長期待。
  • 目前投資評級：分批布局。
  • 主要理由：基本面品質佳、權利金收入具營運槓桿；但估值明顯高於一般半導體設計同業，新手不宜一次追高，較適合等待營收與估值拉回後分批。
  • 主要風險：先進製程導入延遲、主要客戶流片量不如預期、競爭 IP 替代、估值壓縮、地緣政治與晶圓代工景氣循環。
  • 最重要的 3-5 個追蹤指標：月營收、權利金占比、營業利益率、先進製程 tape-out/量產數、主要客戶產品週期。
  • 代表性美股與台股同業：ARM、SNPS、CDNS、2330 台積電、3661 世芯-KY、3443 創意、3035 智原、2454 聯發科、3529 力旺。

2. 公司如何賺錢

  • 主要產品或服務：OTP、MTP、eFlash、PUF/安全相關嵌入式記憶體 IP，提供設計授權、技術服務、矽驗證與量產權利金。
  • 營收來源：技術服務收入與權利金收入；當客戶晶片導入量產後，出貨越多，力旺可收到更多 royalty。
  • 主要客戶：晶圓代工平台、IC 設計公司、ASIC/SoC 開發商、車用/工控/AI 邊緣晶片客戶；個別客戶與占比需查證。
  • 毛利率與營益率特性：IP 授權不需要大量製造資產，毛利率可接近軟體/授權模式；2026Q1 摘要顯示毛利率約 100%、營益率約 60.5%，代表費用控管與 royalty 槓桿強。
  • 成長主要來自價格、出貨量、市占率、新產品，還是產業總需求擴張：長期主要來自先進製程導入、客戶晶片出貨量、IP 新產品與資安需求；短期則受客戶 NRE 與量產權利金認列影響。
  • 是否有景氣循環性：有，但低於製造型半導體。晶片流片與終端需求下滑會延後授權與 royalty；但一旦 IP 進入平台，生命周期可跨多年。

3. 從公司反推產業

  • 公司所屬產業：半導體矽智財、嵌入式非揮發性記憶體 IP、ASIC 設計生態系。
  • 產業解決什麼問題：讓 IC 設計公司不用自行開發高風險記憶體單元與可靠度驗證，縮短晶片上市時間並降低製程轉換風險。
  • 需求來自哪裡：MCU、PMIC、車用電子、網通、AI 加速器、CXL/資料中心控制晶片、資安身分認證與先進製程平台。
  • 產業目前處於導入期、成長期、成熟期，還是景氣循環期：成熟半導體 IP 產業中的結構成長段；先進製程與資安 IP 仍處成長期，但客戶出貨有循環。
  • 長期成長驅動力：晶片客製化、AI ASIC、先進製程成本上升、車用與工控安全需求、IP 重複使用與平台化。
  • 短期波動因素：客戶流片時點、晶圓代工產能、終端需求、匯率、授權案認列節奏與高估值股風險偏好。

5. 商業模式與護城河

  • 技術壁壘：嵌入式 NVM 需要跨製程可靠度、良率、資料保存、耐寫次數與安全驗證，失敗成本高。
  • 規模經濟：IP 可重複授權，既有矽驗證資料越多，後續導入成本越低。
  • 客戶黏著度：一旦客戶晶片採用 IP，轉換供應商需要重新設計、驗證與流片，黏著度高。
  • 轉換成本：高。尤其車用、工控與安全應用重視認證與長期供貨。
  • 品牌或通路優勢：在晶圓代工平台與 IC 設計社群的信任度是重要通路資產。
  • 成本優勢：非製造型資產負擔輕，營收擴張時營業費用率可下降。
  • 專利、法規或資本門檻：專利、製程資料庫、晶圓廠認證、客戶量產紀錄形成門檻。
  • 是否容易被替代：低到中。成熟節點或低階應用可能被替代；先進製程、高可靠度與安全需求替代難度高。

6. 重要財務與營運指標

列出公司與產業最重要的指標，並說明每個指標變好或變壞代表什麼：

  • 營收成長率：2026Q1 摘要顯示營收約 10.9 億元、年增約 19.6%；若月營收持續創高，代表授權與 royalty 同步成長。
  • 毛利率：2026Q1 摘要顯示約 100%；若下降，可能代表低毛利服務案增加或會計分類變化。
  • 營益率：2026Q1 摘要顯示約 60.5%；提升代表 royalty 槓桿變強，下降代表研發/銷售費用擴張或收入組合轉弱。
  • EPS 成長率：2026Q1 EPS 7.98 元，年增約 29%；連續改善支持高估值，若低於預期則估值壓縮風險大。
  • 自由現金流：需查證；IP 公司理論上 FCF 應高於製造業，若應收款或研發資本化異常需留意。
  • 資本支出：通常低於晶圓製造商；資本支出大增需查明是否為測試/研發平台投入。
  • 存貨週轉：IP 公司存貨不是核心指標，應改看應收款、合約負債與 royalty 現金流。
  • 訂單能見度：授權案、晶圓廠平台導入、客戶量產排程是能見度來源。
  • 客戶集中度：若大客戶 royalty 占比過高，單一產品週期會放大波動。
  • 市占率：先進製程 eNVM 與安全 IP 導入數越高，代表平台地位越穩。
  • 產能利用率：不適用於純 IP 公司；可替代觀察研發人力利用率與專案轉量產率。

11. 投資標的分層

  • 核心穩健型：台積電、Synopsys、Cadence；理由是產業地位深、客戶分散且現金流較穩。
  • 成長型：力旺、Arm、世芯-KY；理由是 AI/ASIC/IP 成長性高，但估值敏感。
  • 景氣循環型：部分 IC 設計與 ASIC 設計服務公司；理由是專案與終端需求波動會影響獲利。
  • 高風險高報酬型：高估值、客戶集中或單一題材 IP/ASIC 標的；理由是上修時彈性大，下修時估值壓縮也大。
  • 只適合觀察、不適合輕易買進：缺乏量產驗證、虧損或只有題材沒有現金流的 IP 公司；理由是商業化不確定。

12. 產業風險與反方觀點

  • 估值過高風險：力旺 P/E 100.00，任何成長失速都可能造成倍數壓縮。
  • 景氣反轉風險：晶片客戶若削減流片或終端需求下滑，授權與 royalty 會延後。
  • 技術替代風險：若競爭 IP 在成本、可靠度或先進製程支援勝出，議價力會下降。
  • 競爭加劇風險：國際 EDA/IP 大廠與晶圓廠內建解決方案都可能壓縮空間。
  • 政策與地緣政治風險：半導體出口管制與兩岸風險會影響客戶產品與晶圓代工供應鏈。
  • 客戶集中風險：若少數大客戶貢獻大部分 royalty，產品週期風險放大。
  • 財報可能出現的警訊：月營收連續下滑、營益率低於歷史區間、應收款大增、研發費用無法轉成新授權。
