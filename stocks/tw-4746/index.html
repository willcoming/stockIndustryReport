1. 一頁式懶人包

  • 公司一句話介紹：台耀是台灣 API、HPAPI、ADC 與針劑 CDMO 公司，從原料藥製程開發延伸到高活性藥物與注射劑。
  • 所屬主要產業：生技醫療業、原料藥、學名藥針劑、CDMO、ADC 供應鏈。
  • 產業鏈位置：中游高門檻製程開發與 GMP 製造，兼具部分下游針劑產品商業化。
  • 核心投資邏輯：若 Eribulin、ADC bioconjugation 與 CDMO 案件放量，台耀可從傳統 API 升級為高值化製造平台；但來源 K 線報告列入「明確不可以買 / 暫時排除」，且成交金額低於門檻，短線風險高。
  • 目前投資評級：觀察。
  • 主要理由：公司有真實高門檻製造能力與國際法規認證，但即時估值與 2026 財報證據不足，且來源報告顯示流動性不足與偏空結構。
  • 主要風險：藥證/查廠、單一產品放量不確定、CDMO 接案波動、毛利率受產品組合影響、流動性與股價波動。
  • 最重要的 3-5 個追蹤指標：月營收、毛利率、營業利益率、Eribulin 出貨、ADC/CDMO 訂單與產能利用率。
  • 代表性美股與台股同業：Lonza、Catalent、Thermo Fisher、WuXi AppTec、Samsung Biologics、保瑞、旭富、永信、泰福-KY、台康生技。

2. 公司如何賺錢

  • 主要產品或服務：API、HPAPI、ADC bioconjugation、針劑、客製化合成、發酵與胜肽合成。
  • 營收來源：自有 API/針劑產品銷售、客戶委託開發製造、里程碑或授權相關收益；精確占比需查證。
  • 主要客戶：歐美日藥廠、學名藥公司、生技公司與 CDMO 客戶；公司未在網頁揭露完整客戶占比。
  • 毛利率與營益率特性：一般 API 毛利較受競爭影響；HPAPI、ADC、針劑與高法規門檻 CDMO 若稼動率提升，利潤率可較佳。
  • 成長主要來自價格、出貨量、市占率、新產品，還是產業總需求擴張：短期看 Eribulin 與既有 API 出貨，長期看 ADC/HPAPI/CDMO 案件與產能利用率。
  • 是否有景氣循環性：有部分循環性，但更多受專案進度、藥證、客戶商業化與查廠節奏影響。

3. 從公司反推產業

  • 公司所屬產業：小分子原料藥、針劑學名藥、ADC 製程與生技 CDMO。
  • 產業解決什麼問題：替藥廠提供符合法規的高品質藥物原料、毒性高/技術難製程與商業量產能力。
  • 需求來自哪裡：腫瘤治療、學名藥、專利藥到期、ADC 新藥開發、藥廠外包製造與供應鏈分散。
  • 產業目前處於導入期、成長期、成熟期，還是景氣循環期：一般 API 成熟且競爭激烈；ADC 與高活性 CDMO 處於成長期。
  • 長期成長驅動力：高活性藥物需求、ADC pipeline 擴張、歐美供應鏈韌性、藥廠外包比例提升。
  • 短期波動因素：個別產品核准與出貨、查廠結果、客戶案延遲、匯率、原料成本與股價流動性。

5. 商業模式與護城河

  • 技術壁壘：HPAPI、ADC、注射劑與 cGMP 製程放大形成技術門檻。
  • 規模經濟：公司規模小於國際 CDMO，但利基產品與區域彈性可形成差異。
  • 客戶黏著度：藥品製造一旦完成技轉與法規申報，切換供應商成本高。
  • 轉換成本：高，尤其是已納入藥證文件的 API/製造站點。
  • 品牌或通路優勢：官方資料強調產品銷往歐美日與多國 GMP/法規認證。
  • 成本優勢：相對歐美 CDMO 有成本彈性，但相對中國與印度競爭者未必低。
  • 專利、法規或資本門檻：查廠、DMF、ANDA/BLA、GMP、無菌針劑與高活性設施是核心門檻。
  • 是否容易被替代：一般 API 可替代性高；ADC/HPAPI/針劑平台替代難度較高。

6. 重要財務與營運指標

列出公司與產業最重要的指標，並說明每個指標變好或變壞代表什麼：

  • 營收成長率：若月營收連續年增，代表產品出貨或 CDMO 案件轉收入；若忽高忽低，需確認是否只是批次出貨。
  • 毛利率：高值產品占比提升應推升毛利；若營收增但毛利降，可能是低毛利案或稼動率不足。
  • 營益率：反映固定成本、研發與查廠成本是否被收入吸收。
  • EPS 成長率：需確認獲利是否跟上營收與題材。
  • 自由現金流：擴建與研發可能造成短期壓力，需看營運現金流。
  • 資本支出：HPAPI、ADC、針劑設施投資需轉化為稼動率。
  • 存貨週轉：批次生產與客戶交貨節奏會影響存貨。
  • 訂單能見度：CDMO backlog、商業化批次、授權供應合約。
  • 客戶集中度：需查證，若大客戶延遲會直接衝擊收入。
  • 市占率：利基 API 與 Eribulin 具可觀察性，全球份額需查證。
  • 產能利用率：決定固定成本吸收與毛利率。

12. 產業風險與反方觀點

  • 估值過高風險：ADC/CDMO 題材可能先反映在股價，財報未跟上會壓縮估值。
  • 景氣反轉風險：藥廠庫存調整或客戶延後開發會影響接案。
  • 技術替代風險：國際大廠、印度/中國供應商或客戶內製可能替代部分業務。
  • 競爭加劇風險：一般 API 價格競爭激烈。
  • 政策與地緣政治風險：藥價政策、查廠、供應鏈限制都會影響商業化。
  • 客戶集中風險：需查證，若單一客戶/產品占比高，營收波動會放大。
  • 財報可能出現的警訊：營收成長但毛利下滑、應收帳款上升、存貨增加、營業現金流轉弱。
